# Distribution-Network-Analysis-For-a-UK-based-Electric-Car-Charging-Company
This is a Distribution Network Analysis for a UK-based electric car charging company. Using Python to carry out explorative data analysis- Univariate and Bivariate Analysis 
This is a Distribution Network Analysis for a UK-based electric car charging company. Using Python to carry out explorative data analysis- Univariate and Bivariate Analysis

UNIVARIATE ANALYSIS
1. Visualize the distribution of electricity consumption
2. Analyse the distribution of electric vehicles EV, charging habits, and customer types

BIVARIATE ANALYSIS
1. Use geospatial data to visualize the locations of substations and EV charging stations
2. Analyse the capacity of transmission lines
